%s

You'll see this used throughout the sql queries code. This is a string formatting method; Python replaces the '%s' with the preceding strings. It can be confusing because the % symbol is also used to create "magic commands" in Jupyter notebooks, but in this context it's used as an alias to whatever is in the preceding set
