Pandas Tools

Subsetting Dataframes.
df.query:

Limited to basic queries. Struggles with multiple logical operations.

df.query('col1 > col2')

pd.df.loc

Very fast when selecting rows based on labels.

Better at handling complex queries

df.loc[row_label]
df.loc[[row_label1, row_label2]]
df.loc[df['shield'] > 6]
df.loc[df['shield'] > 6, ['max_speed']] #returns just the 'max_speed' column
df.loc[(df['max_speed'] > 1) & (df['shield'] < 8)]
df.loc[(df['max_speed'] > 4) | (df['shield'] < 5)]

Brackets

Useful for selecting just rows based on complex criteria.

above_35 = titanic[titanic["Age"] > 35]
class_23 = titanic[titanic["Pclass"].isin([2, 3])]
class_23 = titanic[(titanic["Pclass"] == 2) | (titanic["Pclass"] == 3)]

Masking

I think this is just another name for using brackets.
